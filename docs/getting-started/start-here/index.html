Before You Begin

  • Over-ear headphones or both earbuds. Neurima only works when left and right stay separated. Speakers, mono output, or a single earbud will not produce the effect.
  • Local stereo audio already on the device. Neurima does not stream, and the IMA playback path only works on local audio.
  • If you plan to use Music, see Data, Permissions & Media Library for which library tracks qualify.

Choose a Starting Source

Start with audio you already know well. Common first sources:

  • Music: Browse songs already in the device Music library (requires optional Media Library access).
  • Files: Import local audio from the system file picker.
  • Generate: If nothing is ready yet, create a sample ambient track from the playlist add menu.
  • Sound Lab: Make ambient piano tracks in Neurima and send them to the playlist.

First-Session Defaults

  • Processing: on
  • Behavior Mode: Continuous
  • Channel Affinity: Automatic
  • Calibration Profile: default
  • Circadian: off

The default pathways are Relax, Balance, and Focus. You can refine or replace them later in Optimization, Pathways & Circadian.

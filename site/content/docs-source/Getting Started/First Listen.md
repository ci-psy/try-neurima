# First Listen

<- [[Website/Getting Started/_Getting Started|Getting Started]]

## Start Playback

1. Open the playlist add menu and add one track.
2. Press play.
3. Leave the default processing settings in place for the first session.

![Playlist view](_screenshots/playlist-empty.png)

> **Note:** If a `Music` track will not open, see [[Website/Guides/Data, Permissions & Media Library|Data, Permissions & Media Library]].

## Player Basics

![Player view](_screenshots/player-main.png)

| Action | How |
|--------|-----|
| Open Settings | Tap `Settings` |
| Open Playlist | Tap `Playlist` |
| Play or pause | Tap the play/pause button |
| Change the playback display | Double-tap the main display area |

See [[Website/Guides/Player & Playlist|Player & Playlist]].

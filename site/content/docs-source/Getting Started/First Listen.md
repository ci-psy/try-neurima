> **Note:** `Music` works with tracks that are truly local to the device: iTunes Store purchases downloaded to the device or audio synced from the Apple Music app on Mac. Apple Music subscription downloads currently do not work because they remain DRM-protected subscription copies, even when they appear downloaded in the Music app. Purchased iTunes Store tracks do work. If needed, add the file from `Files` instead.

## Player Basics

![Player view](_screenshots/player-main.png)

| Action | How |
|--------|-----|
| Open Settings | Tap `Settings` |
| Open Playlist | Tap `Playlist` |
| Play or pause | Tap the play/pause button |
| Change the playback display | Double-tap the main display area |
| Cycle information views | Tap the center info capsule |
| Set channel affinity to Left / Right | Swipe the center info capsule left or right |
| Return channel affinity to Automatic | Long press the center info capsule |

See [[Website/Guides/Player & Playlist|Player & Playlist]].
